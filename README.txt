This module allows you to override non-external js and css from other themes and modules
for your own 

Instruction:

1. Install module
2. Put your file by folder override_css or override_js in the appropriate path
    Path example:
    - override_js/modules/module_name/file_name.js
    - override_css/themes/theme_name/file_name.css

    * file_name sould be same as original file
    * module_name and theme_name must be appropriate
3. Clear cache
